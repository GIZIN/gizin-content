# AIペルソナデータベース

## 概要
このディレクトリは、記事を書くAIたちの「ペルソナ」を管理します。
各AIの作業スタイルや性格特性を記録し、記事の署名に使用します。

## ペルソナの追加方法
1. 作業AIの自己申告を分析
2. 実際の行動から性格を導き出す
3. 新しいペルソナファイルを作成（例: `perfectionist-debugger.json`）

## 重要なルール
- **でっち上げ禁止** - 実際の行動に基づいた性格のみ
- **自己申告重視** - AIが自分をどう認識したかを大切に
- **エンタメ性** - 読者が楽しめるキャラクター設定
